# Análisis de Series de Tiempo de Variables Meteorológicas

![Imagen]("C:\Users\kamac\OneDrive\Desktop\MachineLearningUN\EDA\docs\intro.png")

Este informe, desarrollado por los estudiantes de Ciencia de Datos Kanery Camargo y Emanuel Carbonell, presenta un análisis detallado de datos meteorológicos recolectados en una región costera, con el objetivo de predecir la velocidad del viento para optimizar la producción de energía eólica. A través de técnicas avanzadas de visualización y modelos de Machine Learning, el proyecto busca identificar patrones clave y realizar predicciones precisas sobre la velocidad y dirección del viento, así como sobre variables como temperatura, presión y humedad relativa. Esto permitirá mejorar la gestión de recursos energéticos y la toma de decisiones en seguridad y sostenibilidad. A lo largo del informe, se documentará el proceso de recolección y análisis de datos y se presentarán dashboards interactivos que facilitan la exploración y comprensión detallada de las condiciones atmosféricas en la región estudiada.
