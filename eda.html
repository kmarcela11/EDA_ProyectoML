Análisis Exploratorio de Datos (EDA)#

En esta parte del proyecto, se realiza un análisis exploratorio de datos (EDA) para comprender las características principales del conjunto de datos meteorológicos. Este análisis incluye la visualización y evaluación de variables clave, como la dirección del viento, la temperatura, la presión y la humedad, con el objetivo de identificar patrones, relaciones y posibles anomalías en los datos. Además, se emplean técnicas descriptivas y gráficas para resumir la distribución y variabilidad de las variables, proporcionando una base sólida para el posterior desarrollo de modelos predictivos y asegurando la calidad de los datos utilizados en el análisis.

Contextualización de los datos#

El conjunto de datos comprende información detallada sobre variables climáticas fundamentales, recopiladas en una región costera mediante un avanzado sistema de monitoreo atmosférico en funcionamiento desde el 1 de enero de 2022. Este sistema cuenta con anemómetros acústicos instalados a alturas de 100 m, 80 m, 60 m y 40 m, que registran la velocidad del viento, complementados con veletas para medir su dirección en los mismos niveles. Además, incluye sensores de temperatura ubicados a 100 m y 20 m, así como dispositivos para medir presión atmosférica y humedad relativa.
Las mediciones se realizan a intervalos de 10 minutos, ofreciendo una resolución temporal elevada y un nivel de detalle que facilita un análisis minucioso de las condiciones atmosféricas. Este diseño de monitoreo proporciona una base robusta para desarrollar modelos predictivos y estudiar las dinámicas del viento en tiempo real, contribuyendo significativamente a la gestión y optimización de recursos energéticos eólicos en la región. El conjunto de datos abarca registros hasta el 16 de mayo de 2022, fecha en que el sistema dejó de operar correctamente.

Reducción de Dimensionalidad#

Para simplificar el análisis y reducir la dimensionalidad del conjunto de datos, se opta por conservar únicamente la variable de promedio (AVG) para cada medición. El promedio proporciona una medida representativa del comportamiento general de las variables (como velocidad del viento, temperatura, presión, etc.) en cada intervalo de 10 minutos. Además, se eliminará la variable de Canal Vacío, ya que esta columna no contiene información relevante o valores de medición que puedan aportar al análisis. Mantenerla solo añadiría ruido al conjunto de datos, complicando innecesariamente el procesamiento y análisis de las variables significativas.

No se aplicará el Factor de Inflación de Varianza (VIF), ya que su uso podría eliminar de forma inconsistente diferentes estadísticas en cada variable, como el promedio en algunas y la desviación estándar en otras. Esto resultaría en un conjunto de datos inconsistente, donde las variables no tendrían el mismo tipo de estadístico representativo, lo cual dificultaría el análisis y la interpretación. Al mantener solo el promedio, se asegura que todas las variables posean un valor consistente y representativo, facilitando una interpretación coherente y un análisis robusto de las tendencias generales.

Análisis por variable#

Métodos a usar#

Antes de proceder con el análisis de cada variable, se desarrollan métodos específicos para simplificar la codificación y optimizar el flujo del análisis, logrando así un proceso más eficiente y organizado. Los métodos creados son los siguientes:

  1. plot_acf_pacf: Genera las gráficas de autocorrelación (ACF) y autocorrelación parcial (PACF) para analizar la dependencia temporal de cada variable.

  2. descomponer_serie: Realiza la descomposición estacional de cada serie temporal, permitiendo visualizar sus componentes de tendencia, estacionalidad y ruido.

  3. plot_histogram_and_boxplot: Genera el histograma y el boxplot de cada variable para examinar su distribución y detectar posibles valores atípicos.

def plot_acf_pacf(data, column, lags = 40, title = ''):
    fig, axes = plt.subplots(1, 2, figsize = (14, 5))
    
    # Gráfico de Autocorrelación (ACF)
    plot_acf(data[column], ax = axes[0], lags = lags)
    axes[0].set_title(f'Autocorrelación (ACF) - {title}')
    
    # Gráfico de Autocorrelación Parcial (PACF)
    plot_pacf(data[column], ax = axes[1], lags = lags)
    axes[1].set_title(f'Autocorrelación Parcial (PACF) - {title}')
    
    plt.tight_layout()
    plt.show()

def descomponer_serie(data, columna, periodo, modelo, titulo, axs=None):
    # Realizar la descomposición estacional
    descomposicion = seasonal_decompose(data[columna], model=modelo, period=periodo)
    
    # Si no se proporcionan ejes, crear una nueva figura
    if axs is None:
        fig, axs = plt.subplots(4, 1, sharex=True, figsize=(10, 8))
    
    # Graficar la serie original, tendencia, estacionalidad y residuales en los ejes proporcionados
    axs[0].plot(descomposicion.observed)
    axs[0].set_ylabel('Original')
    axs[0].set_title(titulo)
    
    axs[1].plot(descomposicion.trend, color='red')
    axs[1].set_ylabel('Tendencia')
    
    axs[2].plot(descomposicion.seasonal, color='green')
    axs[2].set_ylabel('Estacionalidad')
    
    axs[3].plot(descomposicion.resid, color='black')
    axs[3].set_ylabel('Residuales')
    axs[3].set_xlabel('Fecha')

def plot_histogram_and_boxplot(data, column, title):

    fig, axes = plt.subplots(1, 2, figsize=(14, 6))

    # Histograma con KDE
    sns.histplot(data[column], kde=True, color='#473C8B', ax=axes[0])
    axes[0].set_title(f'Distribución de {title}')
    axes[0].set_xlabel(column)
    axes[0].set_ylabel('Frecuencia')

    # Diagrama de cajas horizontal
    sns.boxplot(x=data[column], color='#473C8B', ax=axes[1], orient='h')
    axes[1].set_title(f'Diagrama de Cajas de {title}')
    axes[1].set_xlabel(column)
    axes[1].set_ylabel('')


    plt.tight_layout()
    plt.show()

Prueba de estacionariedad: Dickey-Fuller#

En esta sección, se lleva a cabo la prueba de Dickey-Fuller Aumentada (ADF) para evaluar la estacionariedad de las series de tiempo seleccionadas en el análisis. La estacionariedad es un supuesto crucial en muchos métodos de modelado de series temporales, como ARIMA, y se refiere a la constancia de las propiedades estadísticas de una serie a lo largo del tiempo, es decir, que la media, la varianza y la autocorrelación no cambian con el tiempo.

  • Hipótesis Nula (H₀): La serie de tiempo es NO estacionaria (tiene una raíz unitaria)

  • Hipótesis Alternativa (H₁): La serie de tiempo es estacionaria (no tiene una raíz unitaria)

for i, variable in enumerate(ejex):
    sns.scatterplot(x = data[variable], y = ejey, ax = axs[i], color = 'darkblue')
    axs[i].set_title(f'Scatterplot entre: {variable} y VelViento60m')
    axs[i].set_xlabel(variable)
    axs[i].set_ylabel('vel_viento')

plt.tight_layout()
plt.show()

Los gráficos de dispersión muestran la relación entre la velocidad del viento a 60 m (variable dependiente) y otras variables climáticas como la fecha, dirección del viento, presión, humedad y temperatura. En el caso de la fecha, se observa un patrón cíclico de la velocidad del viento, posiblemente asociado con variaciones estacionales o meteorológicas. La relación con la dirección del viento muestra un comportamiento concentrado alrededor de ciertos ángulos, lo que sugiere una dirección predominante en la región. Las variables de presión y humedad no presentan relaciones lineales claras con la velocidad del viento, aunque la dispersión indica cierta estabilidad en los rangos de medición. Por otro lado, las temperaturas (a 100 m y 21 m) parecen mostrar una relación no lineal inversa, donde velocidades más altas suelen estar asociadas con temperaturas más bajas. Estas observaciones destacan patrones clave que pueden ser útiles para la construcción de modelos predictivos específicos.

for col in ejex.columns:
    if np.issubdtype(data[col].dtype, np.datetime64):
        data[col] = data[col].astype('int64')  

fig, axs = plt.subplots(3, 2, figsize=(18, 12))
axs = axs.flatten()
for i, variable in enumerate(ejex.columns):
    sns.regplot(x=data[variable], y=ejey, ax=axs[i], color='lightblue', 
                scatter_kws={'s': 10}, line_kws={'color': 'darkblue'})
    axs[i].set_title(f'Regplot entre {variable} y VelViento60m')
    axs[i].set_xlabel(variable)

plt.tight_layout()
plt.show()

Los gráficos de regresión muestran tendencias lineales en las relaciones entre la velocidad del viento a 60 m y diversas variables climáticas. Se observa una ligera disminución de la velocidad del viento con el tiempo, reflejando posibles cambios graduales en las condiciones atmosféricas. La dirección del viento muestra una relación negativa moderada, indicando que ciertas direcciones predominantes están asociadas con mayores velocidades. La presión atmosférica también presenta una relación negativa, coherente con sistemas de baja presión que suelen generar vientos más intensos. De manera similar, la humedad relativa muestra una ligera disminución de la velocidad del viento con su aumento, posiblemente debido a condiciones más estables en ambientes húmedos. Finalmente, la temperatura a 100 m y 21 m presenta una relación negativa notable, donde mayores velocidades se asocian con temperaturas más bajas, lo que concuerda con el enfriamiento generado por vientos más rápidos. Estas tendencias respaldan los patrones esperados y servirán como base para construir modelos predictivos más precisos.
